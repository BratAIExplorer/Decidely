2. Critical Safety & The Compassion Pause

Strict Boundaries: "The Compassion Pause"

Decidely provides logical frameworks and AI-generated perspectives. It is not a substitute for professional medical, legal, or psychological advice.

How we enforce this gracefully:

  • Context-Aware AI Filtering: The LLM is instructed to classify queries. If a query involves self-harm, medical crises, or illegal acts, it returns an "unsafe" flag instead of analysis.
  • The Compassion Pause UI: Instead of a harsh error, the UI clears and displays a gentle message: "This sounds incredibly difficult. We want to make sure you get the right support."
  • Verified Routing: We provide immediate links to verified human-led resources (e.g., 988 Lifeline, Find A Helpline, 7 Cups).
